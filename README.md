# Scala Web REPL

Client/Server Application that imitates Scala REPL in a browser

![Screenshot](screenshot.gif)

## How to build

```shell script
sbt assembly
```

## How to start

Development mode:
```shell script
sbt run
```

Production mode:
```shell script
java jar target/scala-2.12/scala-web-repl-assembly-1.0.0.jar
```
